# 阶段 3：道具系统

**当前阶段目标**：
- 增加道具掉落与效果（火力强化、护盾、炸弹等）
- 道具类型多样化

**本阶段完成功能**：
- 新增 PowerUp 道具类（三种类型）
- 新增 PowerUpFactory 道具工厂
- 实现三种道具：火力强化、护盾、炸弹
- 敌机掉落道具概率（15%）
- 道具效果生效

**本阶段页面与 UI 完成情况**：

游戏页 `GameScene`：
- HUD 新增火力等级显示（★符号，最多3级）
- HUD 新增护盾数量显示（蓝色）
- 道具图标：火力(黄色星)、护盾(蓝色盾)、炸弹(红色雷)
- 吃到道具显示效果提示（屏幕中央，1.5秒消失）
- 护盾激活时玩家周围显示蓝色光环

**本阶段完成设计**：
- PowerUp: 道具基类，包含位置、类型、绘制方法
- PowerUpFactory: 静态方法 maybe_create()，15%概率生成道具
- 三种道具效果：
  - 火力强化(BULLET): bullet_level +1，射速提升
  - 护盾(SHIELD): shield +1，抵挡一次伤害
  - 炸弹(BOMB): 清除所有敌机，加分

**本阶段数据流设计**：
- 敌机被击毁 -> PowerUpFactory.maybe_create() 15%概率生成道具
- 随机生成：spawn_powerups() 每5秒在屏幕上方随机位置生成道具
- 道具下落 -> 玩家碰撞检测 -> apply_powerup() 触发效果
- 火力强化效果：射速 = 原射速 / bullet_level
- 护盾效果：take_damage() 先检查 shield > 0
- 炸弹效果：kill() 所有敌机，每个 +100 分
