# 阶段 4：敌机AI与关卡系统

**当前阶段目标**：
- 实现敌机AI行为（追击、包围）
- 波次系统
- Boss战

**本阶段完成功能**：
- 新增敌机AI类型（NORMAL、HOMING、SIDE_ATTACK、BOSS）
- 波次系统：每波敌机数量递增（3 + wave*2），波次间有2秒停顿
- Boss战：每5波出现一次，可发射子弹攻击玩家
- 敌机子弹系统：Boss发射橙色子弹

**本阶段页面与UI完成情况**：

游戏页 `GameScene`：
- HUD新增波次显示（左上角）
- Boss头顶显示红色血条
- 不同敌机类型有不同移动行为

**本阶段完成设计**：

敌机AI类型：
- `AI_NORMAL`: 直线下落（原有行为）
- `AI_HOMING`: 追踪玩家位置移动，高波次出现
- `AI_SIDE`: 横向移动包抄玩家
- `AI_BOSS`: 大型红色三角形，多倍生命，可射击

波次系统：
- `start_next_wave()`: 初始化新波次
- `enemies_in_wave = 3 + wave * 2`: 每波敌机数量
- `wave_break_timer = 2`: 波次间停顿

Boss机制：
- `BOSS_HEALTH = 10`: 10点生命值
- `BOSS_SCORE = 500`: 击败获得500分
- `ENEMY_BULLET_COOLDOWN = 1.5`: 子弹间隔
- `EnemyBullet`: Boss发射的橙色子弹

**本阶段数据流设计**：
- `EnemyFactory.update()`: 管理波次和敌机生成
- `Enemy.update()`: 根据AI类型更新位置
- `Enemy.can_shoot()`: Boss是否可射击
- `GameScene.enemy_shoot()`: Boss发射子弹
- `GameScene.check_collisions()`: 检测敌机子弹与玩家碰撞

**涉及文件**：
- `war_thunder/src/models/enemy.py` - 敌机类，4种AI类型
- `war_thunder/src/models/enemy_factory.py` - 敌机工厂，波次系统
- `war_thunder/src/constants.py` - Boss相关常量
- `war_thunder/src/scenes/game_scene.py` - 游戏主场景

**验证情况**：

- 不同敌机类型移动行为正确
- Boss出现和射击正常
- 波次递增正常
